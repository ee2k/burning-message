개인정보 처리방침

최종 업데이트: 2025년 1월 10일

Burning Message는 사용자의 개인정보 보호를 최우선으로 생각합니다. 본 방침은 귀하의 데이터 처리 방법에 대해 설명합니다.

1. 데이터 수집

사용자가 생성한 콘텐츠(메시지 또는 이미지)는 저장하지 않습니다.

플랫폼 이용을 위해 사용자 계정이나 개인식별정보(PII)가 필요하지 않습니다.

임시 메타데이터(예: IP 주소)는 서비스 기능을 위해서만 처리되며 짧은 기간 후 삭제됩니다.

2. 쿠키 및 추적

사용자 행동을 모니터링하기 위한 쿠키나 추적 메커니즘을 사용하지 않습니다.

3. 데이터 보안

콘텐츠는 전송 중에 암호화됩니다.

이러한 조치에도 불구하고 절대적인 보안을 보장할 수는 없습니다. 사용자는 자신의 책임하에 콘텐츠를 공유하시기 바랍니다.

4. 제3자 접근

법적 요구가 있는 경우를 제외하고 사용자 데이터를 제3자와 공유하지 않습니다.

5. 귀하의 권리

귀하는 데이터 처리 방식을 이해하고 이 약관에 동의하지 않을 경우 서비스 사용을 중단할 권리가 있습니다.

문의사항이나 우려사항이 있으시면 GitHub 정보 페이지의 연락처를 통해 문의해 주시기 바랍니다.
