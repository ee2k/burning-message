隱私權政策

最後更新：2025年1月10日

在 Burning Message，您的隱私是我們的首要考量。本政策說明我們如何處理您的資料。

1. 資料收集

我們不儲存使用者內容（訊息或圖片）。

平台不需要使用者帳戶或個人身分資訊（PII）。

暫時性的元資料（如 IP 位址）僅用於服務營運，並在短時間後刪除。

2. Cookie 和追蹤

我們不使用 Cookie 或追蹤機制來監控使用者行為。

3. 資料安全

內容在傳輸過程中會被加密。

儘管採取了這些措施，我們無法保證絕對的安全性。使用者分享內容時需自行承擔風險。

4. 第三方存取

除非法律要求（如回應法律請求），否則我們不會與第三方共享使用者資料。

5. 您的權利

您有權了解我們如何處理您的資料，如果您不同意這些條款，可以停止使用本服務。

如有問題或疑慮，請透過 GitHub 資訊頁面上的聯絡方式與我們聯繫。
