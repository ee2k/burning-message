隐私政策

最后更新：2025年1月10日

在 Burning Message，您的隐私是我们的首要考虑。本政策说明我们如何处理您的数据。

1. 数据收集

我们不存储用户内容（消息或图片）。

平台不需要用户账户或个人身份信息（PII）。

临时元数据（如 IP 地址）仅用于服务运营，并在短时间后删除。

2. Cookie 和跟踪

我们不使用 Cookie 或跟踪机制来监控用户行为。

3. 数据安全

内容在传输过程中会被加密。

尽管采取了这些措施，我们无法保证绝对的安全性。用户分享内容时需自行承担风险。

4. 第三方访问

除非法律要求（如响应法律请求），否则我们不会与第三方共享用户数据。

5. 您的权利

您有权了解我们如何处理您的数据，如果您不同意这些条款，可以停止使用本服务。

如有问题或疑虑，请通过 GitHub 信息页面上的联系方式与我们联系。
